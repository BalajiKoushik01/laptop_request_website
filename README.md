# 💻 Student Free Laptop Registration Form

Welcome to the **Digital Free Laptop Registration Application**! This is a completely reimagined, world-class administrative form designed to make the laptop application process seamless and visually stunning for students and universities alike.

## ✨ Features

- **🎨 Modern Glassmorphism UI:** A frosted-glass look with a dynamic gradient background.
- **✨ Animated Elements:** Floating blobs, elegant input labels, and a satisfying button interaction.
- **📱 Fully Responsive:** Beautifully adapts to mobile screens and larger displays seamlessly.
- **🚀 Real-time Validation:** Live syntax checks using Regex for every single field with Font Awesome icons.
- **🍞 Custom Toast Notifications:** Replaces ugly browser alerts with an automated, animated success message.
- **⚡ Zero Dependencies:** Built using Pure HTML, CSS, and Vanilla JavaScript. No bulky frameworks required!

## 🛠️ Technologies Used

* **HTML5:** Semantic architecture.
* **CSS3:** Animations (`@keyframes`), custom variables, Flexbox.
* **Vanilla JavaScript:** DOM Manipulation, real-time feedback loops.
* **Google Fonts:** The sleek *Outfit* typography family.
* **Font Awesome:** Integrated vector icons for visual validation feedback.

## 🚀 Getting Started

Since this project operates strictly on client-side code, getting started is practically instant.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/BalajiKoushik01/laptop_request_website.git
   ```
2. **Navigate into the folder:**
   ```bash
   cd laptop_request_website
   ```
3. **Run the file:**
   Simply double-click `index.html` to open it in your favorite web browser!
